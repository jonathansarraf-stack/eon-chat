Setup in 3 steps

1

Install Claude Code

npm install -g @anthropic-ai/claude-code and run claude login

2

Clone and setup

git clone https://github.com/jonathansarraf-stack/eon-chat.git && cd eon-chat && ./setup.sh

3

Activate your license

Paste your license key in the app settings. 14-day free trial included.
